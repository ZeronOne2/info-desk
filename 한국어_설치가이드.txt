Info-Desk 윈도우 설치 가이드 (최종 버전)
==========================================

패키지 호환성 문제 해결됨!

빠른 시작 (권장):
1. ZIP 파일의 모든 파일을 폴더에 압축 해제
2. install.bat 더블클릭 (자동 스마트 설치)
3. run.bat 더블클릭으로 Info-Desk 실행

해결된 문제들:
✅ PyQt5 버전 호환성 문제 해결
✅ 다중 설치 대안 방법 제공
✅ 스마트 패키지 버전 감지
✅ 자동 오류 복구

설치 방법 (추천 순서):

방법 1: 스마트 배치 설치 (최고)
1. install.bat 더블클릭
2. 스크립트가 자동으로 여러 설치 방법 시도:
   - 최신 호환 버전
   - 안정 버전 (5.15.2)
   - 최소 요구사항
   - 수동 설치
3. 완료 후 run.bat 더블클릭

방법 2: PowerShell 스마트 설치
1. install.ps1 우클릭 → "PowerShell에서 실행"
2. 자동 다중 방법 설치
3. python main.py 실행

방법 3: 환경 확인 및 자동 수정
1. python check_environment.py 실행
2. 자동 환경 검증 및 패키지 설치
3. 구성 요소 테스트 포함

방법 4: 수동 설치 (모든 방법 실패 시)
1. 압축 해제된 폴더에서 명령 프롬프트 열기
2. 순서대로 시도:
   pip install -r requirements.txt
   pip install -r requirements_stable.txt
   pip install PyQt5 openpyxl
3. python main.py 실행

포함된 파일들:
├── install.bat              # 스마트 배치 설치 (다중 방법)
├── install.ps1              # PowerShell 스마트 설치
├── run.bat                  # 애플리케이션 실행
├── check_environment.py     # 환경 확인 및 자동 수정
├── requirements.txt         # 최신 호환 버전
├── requirements_stable.txt  # 안정 버전 (백업)
├── requirements_minimal.txt # 최소 요구사항 (대안)
├── main.py                  # 메인 애플리케이션
├── src/                     # 소스 코드
├── config/                  # 설정 파일
└── data/                    # 데이터 저장소

문제 해결:

여전히 패키지 오류가 발생하나요?
1. pip 업데이트: python -m pip install --upgrade pip
2. 안정 버전 시도: pip install -r requirements_stable.txt
3. 수동 설치: pip install PyQt5==5.15.2 openpyxl==3.1.2

Python을 찾을 수 없나요?
- https://python.org 에서 다운로드
- 설치 시 "Add Python to PATH" 체크
- 설치 후 명령 프롬프트 재시작

애플리케이션이 시작되지 않나요?
1. check_environment.py 실행하여 진단
2. ZIP에서 모든 파일이 압축 해제되었는지 확인
3. 압축 해제된 폴더에서 실행하고 있는지 확인

성공 지표:
✓ install.bat이 오류 없이 완료됨
✓ "Installation completed successfully!" 메시지 표시
✓ run.bat으로 애플리케이션 창이 열림
✓ check_environment.py에서 모든 녹색 체크마크 표시

이 버전은 Python이 설치된 모든 Windows 시스템에서 작동해야 합니다!
